## [Unreleased]

### Changed

- Upgraded `wrangler` peer/dev dependency floor to 4.98.0.
- Pointed the dev sidecar's `main` directly at the user entry point now that
  `getPlatformProxy` preserves workflow bindings with `script_name`
  ([cloudflare/workers-sdk#13863](https://github.com/cloudflare/workers-sdk/pull/13863)).
- Reduced `@oselvar/sveltekit-add-worker-exports/hooks` to a no-op handle;
  the export is kept for backwards compatibility but no longer needs to be
  wired into `hooks.server.ts`.

### Removed

- Removed the `./dev-bridge` package export and the internal `src/bridge/`
  HTTP fallback that synthesized Workflow bindings under unpatched
  wrangler/miniflare.
